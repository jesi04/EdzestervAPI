Request method url path web service authorization
GET Felhasználók A Felhasználók névsora engedélyezett

GET Feladatok A feladatok listája engedélyezett
GET feladatok/{id} Egy kiválasztott feladat adatai engedélyezett
GET Célok A célok listája engedélyezett
GET celok/{id} Egy kiválasztott cél adatai engedélyezett
GET Bemutatók A bemutatók listája engedélyezett
GET bemutatok/{id} Egy kiválasztott bemutató adatai engedélyezett
GET Javasolt Edzések A javasolt edzések listája engedélyezett
GET javasoltak/{id} Egy kiválasztott javasolt edzés adatai engedélyezett
POST felhasznalok Új felhasználó felvitele jelszóval védendő
PATCH felhasznalok/{id} Felhasználó emailjének a módosítása jelszóval védendő
DELETE felhasznalok/{id} Kiválasztott felhasználó törlése jelszóval védendő

DELETE feladatok/{id} Kiválasztott feladat törlése jelszóval védendő

DELETE celok/{id} Kiválasztott cél törlése jelszóval védendő
POST bemutatok Új bemutató felvitele jelszóval védendő
PATCH bemutatok/{id} Bemutató fájl nevének a módosítása jelszóval védendő
DELETE bemutatok/{id} Kiválasztott bemutató törlése jelszóval védendő
POST javasoltak Új javasolt edzés felvitele jelszóval védendő
